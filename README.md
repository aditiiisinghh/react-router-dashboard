# 🚀 React Router Dashboard

A sleek, modern multi-page dashboard application built with **React Router** and **Tailwind CSS**. This project demonstrates clean routing architecture, reusable components, and responsive UI design.

---

## 📸 Preview

![App Preview](https://via.placeholder.com/1000x500?text=React+Router+Dashboard)

---

## ✨ Features

* ⚡ Multi-page navigation using React Router
* 🎨 Styled with Tailwind CSS
* 📱 Fully responsive layout
* 🧩 Reusable component structure
* 🧭 Dynamic navigation header
* 📄 Pages included:

  * Home
  * Support
  * Labs
  * About

---

## 🛠️ Tech Stack

* **React**
* **React Router**
* **Tailwind CSS**
* **JavaScript (ES6+)**

## ⚙️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/react-router-dashboard.git
cd react-router-dashboard
npm install
```

---

## ▶️ Run Locally

```bash
npm run dev
```

Then open:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```
